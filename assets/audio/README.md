# Audio Assets

## Adding Your Wrapped Music

Place your music file here with the name: **`wrapped-music.mp3`**

### Supported formats:
- `.mp3` (recommended)
- `.m4a`
- `.wav`
- `.aac`

### Recommendations:
- Keep file size under 5MB for faster loading
- Use a 30-60 second loop for best experience
- Volume will be set to 30% by default (adjustable in code)
- The music will loop continuously while viewing Wrapped

### Finding free music:
- [Pixabay Music](https://pixabay.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)

Make sure you have the rights to use any music you add!
